# quranprism docs

Multilingual Quran listening. One source text refracted into every language a listener wants — verse by verse.

## Sections

- **[Product Requirements](prd/README.md)** — what the product must do and why, per domain, with stable requirement IDs. Source of truth lives in `prd/`.
- **Engineering** — architecture, database schema. Decisions are recorded as ADRs in `docs/decisions/`.
- **Legal** — privacy policy, terms of service.

> Product/business rules → `prd/`. Engineering "how" → `docs/decisions/`. Current work → `tasks/`.
